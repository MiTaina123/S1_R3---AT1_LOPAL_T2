 Relatório de Atividades  

Objetivo: Este projeto teve como foco consolidar os fundamentos da linguagem Python, explorando desde a manipulação de estruturas de repetição até o gerenciamento de coleções de dados (listas e dicionários). 

O que foi desenvolvido: 

Lógica de Repetição: Utilizei os laços for e while para resolver os problemas, como a sequência de Fibonacci, cálculos de fatorial e geração de tabuadas. 

Manipulação de Dados: Trabalhamos com listas para organizar e extrair informações (máximo, mínimo etc.) e dicionários para simular o armazenamento de tabelas estruturadas. 

Validação e Segurança: Criei scripts capazes de validar entradas de usuários (como formulários de cadastro) e sistemas simples de autenticação por senha, garantindo que o programa só prossiga com dados corretos. 

Tratamento de Strings: Explorei a estética e manipulação de textos com o exercício de impressão vertical em escada. 

 Ferramentas Utilizadas: O desenvolvimento foi realizado utilizando Visual Studio Code e Google Colab; usei o VS CODE para subir para o Github os comandos que fiz no Google Colab. 

Conclusão: A série de exercícios permitiu uma visão prática de como o Python lida com problemas do dia a dia de um desenvolvedor: validar dados, organizar listas e automatizar cálculos matemáticos de forma eficiente.
